1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

 * Variables // Es la forma de almacenar el valor de algo para usarlo mas tarde.
 * Strings // Bloques de textos los cuales se define entre comillas.
 * Funciones (argumentos, `return`) // Serie de pasos a cumplir para obtener informacion de regreso.
 * Declaraciones `if` // Condiciones que se aplican al contexto de una funcion para poder elegir un camino especifico al cual se requeria llegar.
 * Valores booleanos (`true`, `false`) // son aquellos que nos ayudan a reconocer si una instruccion brindada se cumple correctamente o es falsa su comparacion.
